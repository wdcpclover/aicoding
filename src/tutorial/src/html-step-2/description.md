# 常用标签 {#html-step-2}

网页内容主要靠下面这几类标签搭起来。

## 标题与段落

```html
<h1> ~ <h6>   <!-- 六级标题，h1 最大，h6 最小 -->
<p>           <!-- 段落 -->
```

## 列表

```html
<ul>              <!-- 无序列表：每项前面是圆点 -->
  <li>列表项</li>
</ul>

<ol>              <!-- 有序列表：每项前面是数字 -->
  <li>第一项</li>
</ol>
```

`<li>`（list item）是列表里的每一项，必须放在 `<ul>` 或 `<ol>` 里。

## 链接与图片

```html
<a href="https://cn.vuejs.org">这是一个链接</a>
<img src="图片地址" alt="图片说明">
```

- `<a>` 的 `href` 是要跳转去的网址
- `<img>` 的 `src` 是图片地址，`alt` 是图片加载不出来时显示的替代文字
