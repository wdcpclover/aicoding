五、v-if vs v-show 对比

v-if 区块：

✅ 存在于 DOM
❌ 已从 DOM 移除

v-show 区块：

✅ 存在于 DOM
🙈 元素还在，只是 display: none
📝 选择原则：频繁切换用 v-show（如 Tab），很少变化用 v-if（如权限控制）
