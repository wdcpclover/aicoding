一、声明响应式状态

count: {{ count }}

message: {{ message }}

💡 试试修改右侧代码中的初始值，然后观察这里的变化

二、ref() vs reactive()methods 方法

state.name: {{ state.name }}

state.version: {{ state.version }}

💡 reactive 对象直接修改属性，不需要 .value methods 中通过 this.state 访问和修改

三、深层响应性

obj.nested.count: {{ obj.nested.count }}

obj.arr: [{{ obj.arr.join(', ') }}]

💡 嵌套对象和数组的变化也能被自动检测到

四、DOM 更新时机 (nextTick)

消息: {{ domMsg }}

{{ log.text }}

💡 点击按钮，观察"修改后立即读取"和"nextTick 后读取"的区别

五、模板表达式

count + 1 = {{ count + 1 }}

message 反转 = {{ message.split('').reverse().join('') }}

message 长度 = {{ message.length }}

💡 模板中可以直接写 JS 表达式，响应式数据变化时自动重新求值
