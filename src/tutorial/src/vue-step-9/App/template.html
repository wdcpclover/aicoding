一、手动绑定 vs v-model

值："{{ text }}"

💡 两个输入框绑定同一个变量，改一个另一个也变——它们完全等价

二、文本 & 多行文本

message："{{ message }}"

{{ multilineText || '（预览区）' }}

三、复选框

单个复选框 → 布尔值

agreed：{{ agreed }}

✅ 已同意，可以继续

多个复选框 → 数组

checkedFruits：[{{ checkedFruits.join(', ') }}]

💡 多个 checkbox 绑定同一个数组，勾选/取消自动添加/移除 value

四、单选按钮

pickedSize："{{ pickedSize }}"

示例文字 — 字号随选择变化

五、选择器

selected："{{ selected }}"

dynamicSelected："{{ dynamicSelected }}"

六、值绑定

自定义 true-value / false-value

toggle："{{ toggle }}" （{{ toggle === 'yes' ? '已订阅' : '未订阅' }}）

💡 不是 true/false，而是 "yes"/"no"——通过 true-value/false-value 自定义

select 绑定对象值

selectedPerson：{{ JSON.stringify(selectedPerson) }}

💡 :value 绑定对象后，选中的值就是整个对象，而不只是字符串

七、修饰符

.lazy — 失焦后才同步

"{{ noLazyMsg }}"

"{{ lazyMsg }}"

.number — 自动转数字

{{ ageStr }} + 1 = {{ ageStr + 1 }} ({{ typeof ageStr }})

{{ ageNum }} + 1 = {{ ageNum + 1 }} ({{ typeof ageNum }})

.trim — 去除首尾空格

值："{{ trimMsg }}" (长度: {{ trimMsg.length }})

💡 对比 .number：左边 "25" + 1 = "251"（字符串拼接），右边 25 + 1 = 26（数字相加）
