# 盒模型 {#css-step-2}

CSS 里**每个元素都是一个盒子**。理解盒模型，是搞懂布局的第一步。

## 一个盒子由四层组成

```
┌─────────────────────────────┐  ← margin（外边距，盒子和别人的距离）
│  ┌───────────────────────┐  │  ← border（边框）
│  │  ┌─────────────────┐  │  │  ← padding（内边距，内容和边框的距离）
│  │  │     content     │  │  │  ← content（内容）
│  │  └─────────────────┘  │  │
│  └───────────────────────┘  │
└─────────────────────────────┘
```

```css
.box {
  width: 200px;          /* 内容宽度 */
  padding: 20px;         /* 内边距：内容撑开 */
  border: 2px solid #42b883;  /* 边框：粗细 样式 颜色 */
  margin: 16px;          /* 外边距：和其他盒子隔开 */
}
```

## 一个坑：box-sizing

默认情况下，`width` 只算 content，加了 padding 和 border 后盒子会**变得比 width 更大**。加上这一行就能让 `width` 包含 padding 和 border，省心：

```css
* { box-sizing: border-box; }
```

几乎所有项目都会写这行，记住它。
