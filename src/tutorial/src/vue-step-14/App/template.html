一、基础事件触发

📨 父组件收到消息：{{ childMsg }}

💡 子组件创建时自动 emit('response', ...) → 父组件通过 @response 接收

二、交互触发事件

父组件计数：{{ count }}

💡 点击上方子组件中的 +1 / +5 按钮，子组件 emit → 父组件更新计数

三、传递多个参数

📋 父组件收到提交结果：

姓名：{{ submitResult.name }}

分数：{{ submitResult.score }}

等级：{{ submitResult.level }}

等待子组件提交数据...

💡 填写上方子组件中的表单并提交，emit 可传递多个参数

四、v-model 双向绑定

父组件中的 text：{{ text }}

💡 在子组件输入框中修改内容，父组件数据实时同步。v-model 就是 :modelValue + @update:modelValue 的简写
