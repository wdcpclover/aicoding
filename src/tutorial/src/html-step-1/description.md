# 第一个网页 {#html-step-1}

HTML（HyperText Markup Language，超文本标记语言）是网页的**骨架**。它用一对对**标签**告诉浏览器：哪里是标题、哪里是段落、哪里是图片。

## 标签长什么样

```html
<h1>我是一级标题</h1>
<p>我是一段文字。</p>
```

- `<h1>` 是**开始标签**，`</h1>` 是**结束标签**，中间夹着内容
- 开始标签 + 内容 + 结束标签，合起来叫一个**元素**

## 一个网页的基本结构

```html
<!DOCTYPE html>            <!-- 声明这是 HTML5 文档 -->
<html>
  <head>                   <!-- 头部：给浏览器看的信息 -->
    <meta charset="UTF-8"> <!-- 用 UTF-8，中文不乱码 -->
    <title>标题</title>     <!-- 浏览器标签页上的名字 -->
  </head>
  <body>                   <!-- 身体：用户真正看到的内容 -->
    <h1>你好</h1>
  </body>
</html>
```

记住一句话：**`<head>` 里是写给浏览器的，`<body>` 里才是给人看的。**
