一、Class 绑定 - 对象语法

class="static" + :class="{ active: {{ isActive }}, 'text-danger': {{ hasError }} }"

二、Class 绑定 - 计算属性对象

:class="classObject" (computed)

三、Class 绑定 - 数组语法

:class="['{{ activeClass }}', '{{ errorClass }}']"

数组中嵌套三元：:class="[isActive ? activeClass : '', errorClass]"

四、Style 绑定 - 对象语法

:style="{ color: '{{ activeColor }}', fontSize: '{{ fontSize }}px' }"

五、Style 绑定 - 样式对象

:style="styleObject" → color + fontSize + fontWeight

六、Style 绑定 - 数组语法

:style="[baseStyles, overridingStyles]" → 合并多个样式对象
