一、v-for 遍历数组 — Todo List

  • {{ index }}. {{ todo.text }}

列表为空，添加一些待办吧！

💡 注意 v-for 同时拿到了 (todo, index) 两个参数，:key 用的是 todo.id

二、v-for 遍历对象

  • {{ index }}. {{ key }}: {{ value }}

💡 遍历对象可拿到三个参数：(value, key, index)

三、key 的重要性

⚠️ 先在下方输入框里各输入不同的内容，然后点击"在开头插入"观察差异

❌ key=index

{{ item.label }}

✅ key=id

{{ item.label }}

💡 左边输入内容会"串位"（index 变了，DOM 复用出错），右边正确跟随（id 不变）

四、数组变更方法

数组：[{{ fruits.join(', ') }}]

💡 这些方法直接修改原数组，Vue 会自动检测变化并更新视图

五、过滤与排序（计算属性）

原始数据：{{ numbers.join(', ') }}

偶数过滤： {{ n }} 无

排序结果： {{ n }}

💡 用 computed 过滤/排序，原始数据始终不变（对比上方数组变更方法的 reverse/sort 会改原数组）
